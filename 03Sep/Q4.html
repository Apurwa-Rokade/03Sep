HTML Tags Information

  1. <html>

    The root element that wraps the entire HTML document.

  2. <head>

    Contains metadata about the document, such as title, links to stylesheets, scripts, and other meta-information.

  3. <title>

    Defines the title of the HTML document displayed in the browser's title bar or tab.

  4. <body>

    Contains the visible content of the HTML document, including text, images, links, and other elements.

  5. <h1> to <h6>

    Heading tags used to define headings of different levels, indicating the hierarchical structure of the content.

  6. <p>

    Defines a paragraph of text, used to structure and separate blocks of text.

  7. <a>

    Creates hyperlinks to other web pages or resources for navigation.
